### 🗂️ High-Impact Projects

#### 🌾 [KrishiSetu: AI Vernacular Assistant for Farmers](https://github.com/RatishPatil37)
*   **Computer Vision Pipeline:** Trained a custom **YOLOv8 classification model** on 22K+ images to diagnose 24 distinct crop diseases with **98% accuracy**, optimized heavily for lightweight CPU inference[cite: 1].
*   **Multimodal NLP & RAG:** Engineered a multilingual voice assistant utilizing a **RAG workflow** and Tesseract OCR to parse land records (7/12 docs) and dynamically rank scheme eligibility[cite: 1].
*   **Data Resiliency:** Deployed webhooks via SMS to dispatch real-time health reports and weather forecasts to low or zero-connectivity rural zones[cite: 1].
*   *Stack: Node.js, React, FastAPI, Gemini API, YOLOv8-cls, HuggingFace, Docker*[cite: 1]

#### 👓 [Smart Sight: AI Navigation for Visual Impairment](https://github.com/RatishPatil37)
*   **Edge-AI System:** Built a standalone, multimodal assistant running on a **Raspberry Pi 4B**[cite: 1]. Deployed an optimized **YOLOv4-tiny** and OpenCV pipeline sustaining a stable **10 FPS**[cite: 1].
*   **Offline Voice Agent:** Implemented a fully private local agent using **TinyLlama** and **faster_whisper** to process intent mapping entirely on-device without cloud dependencies[cite: 1].
*   *Stack: Raspberry Pi 4B, YOLOv4-tiny, OpenCV, Tesseract OCR, Ollama, Python*[cite: 1]

#### 📈 [Review Insight Navigator](https://github.com/RatishPatil37)
*   **Scraping Automation:** Automated dynamic browser data extraction for 30+ product listings via **Selenium** and **BeautifulSoup**[cite: 1].
*   **Sentiment Classifier:** Applied an NLP pipeline yielding **85% accuracy** to cluster user opinions into clear market indicators, visualized over an interactive **Streamlit dashboard**[cite: 1].
*   *Stack: Python, Scikit-learn, Selenium, BeautifulSoup, Streamlit*[cite: 1]
